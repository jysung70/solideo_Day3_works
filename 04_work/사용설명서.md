# 사용 설명서

비밀번호 안전성을 확인하기 위한 간단한 웹 페이지 프로젝트입니다. 페이지 구조와 사용 방법은 아래 내용을 참고하세요.

## 실행 방법
1. `index.html` 파일을 더블클릭하거나 브라우저 `파일 > 열기` 메뉴로 불러옵니다.
2. 페이지가 열리면 비밀번호 입력창과 결과 영역이 표시됩니다.
3. 비밀번호를 입력한 뒤 버튼을 클릭하면 검사 결과가 화면에 출력됩니다.

## 주요 파일 설명
- `index.html`: 입력 폼, 오류 메시지 영역 등 UI 구조와 기본 콘텐츠를 정의합니다.
- `style.css`: 전체 레이아웃과 색상, 버튼/입력창 스타일, 반응형 요소를 지정합니다.
- `script.js`: 비밀번호 체크 로직(길이, 특수문자, 중복 여부 등)과 화면 표시 제어를 담당합니다.
- `passwords_to_check.txt`: 금지된 비밀번호 목록을 관리하며, 필요 시 업데이트합니다.

## 기타
- 이미지 파일 `화면 캡처.png`는 참고 자료입니다.
- 새로운 요구사항이나 검증 규칙을 추가하려면 `script.js`를 수정한 후 브라우저를 새로고침합니다.
- 스타일 변경 시 `style.css`를 수정하고, 저장 후 페이지를 다시 로드하면 즉시 반영됩니다.
