## 📘 텍스트 분석기 사용 설명서

### 1. 개요
- `text_analyzer.html` 하나만 열면 동작하는 브라우저 기반 텍스트 통계 툴입니다.
- `style.css`, `script.js`, `sample_text.txt`(옵션)를 동일 폴더에 두면 스타일과 스크립트가 연결됩니다.
- 인터넷 연결이나 추가 설치가 필요 없으며 PC·모바일 최신 브라우저(크롬/엣지/사파리 등)에서 사용 가능합니다.

### 2. 실행 방법
1. 파일 탐색기에서 프로젝트 폴더를 연 뒤 `text_analyzer.html`을 더블클릭하거나 브라우저로 끌어다 놓습니다.
2. 페이지가 열리면 상단 헤더 아래에 업로드 영역과 텍스트 입력 영역이 표시됩니다.

### 3. 텍스트 입력 방법
#### (1) 텍스트 파일 업로드
- `텍스트 파일 선택` 버튼을 클릭하여 `.txt` 파일을 선택합니다.
- 파일명이 `선택된 파일 없음` 영역에 표시되고 해당 파일 내용이 자동으로 입력창에 로드되어 즉시 분석됩니다.
- drag & drop도 지원: 업로드 영역으로 `.txt` 파일을 끌어다 놓으면 같은 방식으로 처리됩니다. 다른 확장자를 놓으면 “텍스트 파일(.txt)만 업로드 가능합니다.” 알림이 뜹니다.

#### (2) 직접 입력
- `또는 직접 텍스트 입력` 영역에 텍스트를 붙여넣거나 작성합니다.
- 입력 후 `분석하기` 버튼을 누르면 분석이 실행됩니다. 공백만 있거나 비어 있으면 안내 알림이 뜹니다.

### 4. 분석 결과 항목
분석이 완료되면 페이지 하단 `📈 분석 결과` 섹션이 나타나며 자동으로 스크롤됩니다.
- **총 문자 수**: 공백 포함 문자 개수
- **문자 수 (공백 제외)**: 공백 제거 후 문자 개수
- **단어 수**: 공백으로 분리된 한글/영문 단어 수
- **문장 수**: `. ! ? 。` 등 종결 부호 기준 문장 개수
- **한글 문자 / 영문 문자 / 숫자**: 각 문자 유형의 개수
- **줄 수**: 줄바꿈 기준 라인 수
- **평균 단어 길이**: 공백 제외 문자 수 ÷ 단어 수 (단위: 문자)
- **평균 문장 길이**: 단어 수 ÷ 문장 수 (단위: 단어)
- **한글 비율 / 영문 비율**: 한글·영문 문자 비중(%)
- 모든 수치는 천 단위 구분 기호와 단위를 함께 표시합니다.

### 5. 자주 묻는 사항
- **파일이 바로 열리지 않아요.**  
  브라우저 팝업 차단이나 로컬 파일 접근 제한이 있는 경우 강력 새로고침(Ctrl+Shift+R) 또는 다른 브라우저를 사용하세요.
- **긴 텍스트를 분석하면 느려질 수 있나요?**  
  브라우저 메모리 범위 내에서는 대부분 즉시 처리됩니다. 매우 큰 파일(수십 MB)이라면 직접 입력 대신 파일 업로드를 권장합니다.
- **결과를 다시 보고 싶을 때는?**  
  동일한 입력으로 다시 `분석하기`를 누르거나 다른 파일/텍스트를 입력하면 결과가 갱신됩니다.

### 6. 기타 팁
- `sample_text.txt`를 함께 배치하면 예시 텍스트로 기능을 빠르게 확인할 수 있습니다.
- 모바일에서 파일 업로드가 어렵다면 클라우드 노트에 텍스트를 복사한 뒤 직접 입력 방식을 사용하세요.
- 결과 값을 다른 문서에 옮기고 싶다면 숫자를 복사하여 보고서나 메모에 붙여넣으면 됩니다.
